Blue Palette Classes

Blue color palette contains 13 colors: 1 main blue color and 4 lighten colors, 4 darken colors and 4 Accent colors. All colors works perfectly with any bootstrap components, form components or any modern controls, that makes Modern very effective, flexible and configurable.

Please note: You can use .bg-blue and .btn-blue as main colors, but if you want to use lighten, darken or accent colors, use bg-* bg-lighten-NUMBER for lighten color, bg-* bg-darken-NUMBER for darken color and bg-* bg-accent-NUMBER for accent colors.

Class Description
.bg-blue Blue Class for background color.

.bg-lighten-*		 For lighten blue background color, this two classes needed. Here *: 1,2,3,4 lighten blue color options.
