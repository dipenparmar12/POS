Description

For light template, Modern Admin provides light color vertical navigation menu options, you can check light color vertical navigation menu on left side.

Experience it!

This page contain light navigation menu options example, check at the left hand side of the page.

CSS Classes

This table contains all classes related to the light vertical navigation option.

All these options can be set via following classes:

Classes Description
.menu-light To create light vertical navigation, you need to add .menu-light class in navigation <div> tag. Refer HTML markup line no 12.

HTML Markup

This section contains HTML Markup to create light vertical navigation. This markup define where to add css classes to make vertical navigation light.

  • Line no 12: Contain the .menu-light class to set vertical navigation color light.

HPUG Configuration

Modern Admin use PUG as template engine to generate pages and whole template quickly using node js, for getting start with PUG usage & template generating process please refer template documentation.

PUG Variables

This table contains required PUG variables to generate vertical navigation light.

Block Variable Datatype Value Description
verticalMenuConfig menuColor string 'menu-light' verticalMenuConfig block has a menuColor variable, it contains navigation color specific classes in that you need to add 'menu-light'
Modern Admin Template documentation has full list of all blocks, datatype, variables and their description table. Please review it for more configuration options.
PUG Code

Use following PUG code to generate page or template having vertical light navigation.

  • Line no 20-21: Set vertical light navigation class to menuColor variable value as 'menu-light' in verticalMenuConfig block.
  • Line no 27-28: content block has content section html file included include ../contents/vertical-light-navigation.html, which can be customizable on page level.

If you want to use vertical light navigation options on page level you need to define it on page it self. This template has one example pug file for vertical-light-navigation.pug, however you can use it on template level but it will generate whole template have vertical light navigation.

              
                block pageVars
                    - var pageTitle    = "Navigation Light"
                    - var pageSubTitle = "Light color navigation menu"
                    - var description  = "The vertical light navigation light template usage & customization option."
                    - var activeMenu   = "vertical-nav-light"

                extends template

                append verticalMenuConfig
                    - var menuColor = 'menu-light'

                append breadcrumbs
                    +breadcrumbs([{url:"index.html",name:"Home"},{url:"#",name:"Vertical Navigation"}, {name:"Navigation Light"}])

                //- Include page content in page block
                append content
                    include ../contents/vertical-nav-light.html

                //- Vendor CSS
                //------------------------------
                //- Add Vendor specific CSS
                append vendorcss

                //- Page CSS
                //------------------------------
                //- Add custom page specific CSS
                append pagecss

                //- Vendor JS
                //------------------------------
                //- Add vendor specific JS
                append vendorjs

                //- Page JS
                //------------------------------
                //- Add custom page specific JS
                append pagejs
